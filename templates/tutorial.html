Getting Started!

Signing Up:

At the top right of the screen within the header there are Sign Up / Login buttons, use these to create an account!

If you're on a smaller screen, there will be a dropdown button, when the menu opens the Sign Up / Login buttons will be at the bottom of it.

Forum:

Once signed in, you can navigate to the GuildBoard page using the header. Once there you will see other posts made by users and have the oppurtunity to make your own posts!

On the right there will be a 'Create Post' button to get started!

Chatroom Commands:

!roll #: Generates a random number between 1 and the specified limit (e.g., !roll 20).

!add string: Adds items to the inventory (e.g., !add sword or !add sword,,shield). Supports multiple items separated by two commas: ',,' .

!remove #: Removes one item from the inventory based on the provided index.

Gameplay:

Once signed in, you can navigate to the Play Now page using the header. This will allow you to view your active game sessions, create a new session or join an already made session.

When creating a game, you will be able to type in what game the session is for, as well as give the session a title, image and description. Make sure to make it descriptive and to use the forum to help find players to join you on your next campaign!
